# JavaScript Problem Solving

A collection of JavaScript practice files covering fundamental concepts and problem-solving exercises. Each file focuses on a specific topic with multiple tasks and examples.

## Topics Covered

| File                | Topic                                                                                                                                     |
| ------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| `practice.js`       | Basics — arithmetic, conditionals, `isNaN`, `toFixed`                                                                                     |
| `forLoop.js`        | `for` loops — odd/even numbers, sums, multiplication table, `break`/`continue`                                                            |
| `whileLoop.js`      | `while` loops — same patterns as for-loop, closure with `setTimeout`                                                                      |
| `nestedif.js`       | Nested `if-else` conditions                                                                                                               |
| `result.js`         | Grade checker using `if-else if`                                                                                                          |
| `bmi.js`            | BMI calculator with weight/height in imperial units                                                                                       |
| `bus.js`            | Bus fare calculator with age-based discounts                                                                                              |
| `toFixed.js`        | Floating point precision with `toFixed()`                                                                                                 |
| `string.js`         | String manipulation — count characters, vowel check, character swap, title case                                                           |
| `array.js`          | Array basics — access, `push`/`pop`, `includes`, `isArray`, `concat`                                                                      |
| `array-looping.js`  | Array looping — reverse array, filter evens, string concat, reverse sentence, spread copy, 2D arrays                                      |
| `func.js`           | Functions — multiply, odd/even check, average, count zeros in binary string, vowel counter, longest word, random numbers, employee salary |
| `object.js`         | Objects — property access, nested objects, `Object.keys`/`values`/`entries`, `for...in` loop                                              |
| `ProblemSolving.js` | Finding the smallest number in an array using `Math.min`                                                                                  |
| `propagation.js`    | DOM event propagation and `stopPropagation()`                                                                                             |
| `propagation.html`  | HTML page for the event propagation demo                                                                                                  |

## Concepts Practiced

- Variables, data types, and operators
- Conditional statements (`if`, `else if`, `else`, nested)
- Loops (`for`, `while`, `for...of`, `for...in`, `break`, `continue`)
- Functions (parameters, return values, rest parameters)
- Arrays (CRUD operations, iteration, spread operator, 2D arrays)
- Objects (access patterns, built-in methods, iteration)
- String manipulation
- Math methods (`Math.min`, `Math.random`, `Math.round`)
- DOM events and event propagation

## How to Run

You need [Node.js](https://nodejs.org) installed to run the files locally.

```bash
node filename.js
# example
node bmi.js
```

For the event propagation demo, open `propagation.html` directly in a browser.

## Goals

- Build a strong foundation in JavaScript fundamentals
- Practice problem solving with real-world style tasks
- Understand how JavaScript handles numbers, strings, arrays, and objects
